What I've Made

Home Finder

Project Link GitHub
Built with a React frontend, Nodes.js and Express on the backend. A Postgres database stores everything with Backbone.js providing the structure. Sass for styling, authentication done using Passport and Bcrypt and all compiled with Gulp. Allows creation of user accounts who can then post a profile and property listings using a custom admin dashboard. Guest users can view profiles and properties. Mobile-first, responsive design. Photo uploads handled via FileStack. App layout is done using a custom CSS grid system.

Texas Legislature Countdown Clock

Project Link GitHub
A project made for practical self-serving reasons as a legislative staffer at the Texas Capitol. A JavaScript-heavy site that proides for a real-time countdown clock for both the beginning and end of Texas legislative sessions and a calendar showing all legislative deadlines and state & federal holidays. Date and time calculations are helped by the date-fns library, the layout is mostly done with a combination of CSS Grid and Flexbox. It is build on a simple Gulp framework and Sass. Includes a favicon package with a site.webmanifest to allow the site to operated as a progressive web app.

Buda Fooda

Project Link GitHub
